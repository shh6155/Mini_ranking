
# ***Mini_ranking***
> 모든 랭킹을 한자리에!!.  

팝업창을 이용한 여러 사이트들의 랭킹을 모음

## 컨셉
메인창에서 각 사이트 버튼을 클릭시 팝업창을 통해 그 사이트의 주요랭킹 표시





## 사용 예제

## 업데이트 내역
*6.1.1  
   *멜론,지니 실시간 순위 크롤링 구현  
   *디테일 작업   
* 5.30.1  
   * Css 작업 중
* 5.30.1  
   * 네이버,다음 실시간 순위 크롤링 구현
   * 버튼 클릭시 각각의 순위 팝업창 출력
* 5.29.1
   * 첫  컨셉 모델링 시작
